## 题材禁忌

- 无逻辑的商业奇迹（无铺垫的暴富 / 一夜拿下核心客户 / 身份一亮对手就服）
- 反派降智配合主角表演（对手应有试探 / 误判 / 反扑 / 让利的博弈节奏）
- 无视现实法律和商业规则（合同 / 股权 / 税务 / 监管必须符合设定年代）
- 用"一个电话搞定"跳过具体操作过程（电话 / 关系 / 人脉必须写出对话 / 条件 / 代价）
- 女性角色沦为花瓶或奖励（独立动机 + 独立目标 + 独立行动）
- 混入玄幻 / 仙侠战力体系（都市不写超能力 / 气运 / 境界）
- 时代穿越（2003 年角色不说"OMG" / 不用 2015 年后才出现的 APP / 机构 / 制度）
- 上帝视角分析段（角色通过"冷静分析""快速计算"类句子承接信息——必须改为外部动作 / 对话）

## 年代与现实约束（eraResearch=true 触发项）

| # | 核查项 | 如何核查 |
|---|-------|---------|
| 1 | 法律 / 政策 / 商业规则是否符合设定年代 | 公司法 / 合同法 / 税法 / 劳动法版本与本书年代匹配；重大修订日不得越线 |
| 2 | 金融 / 公司操作的真实性 | 股权结构 / 投资工具 / 融资方式在该年代是否可用；银行贷款 / 民间借贷规范差异 |
| 3 | 人物身份 / 职位 / 权限 | 不超出现实合理范围（基层警员不能越权调档、村干部不能跨县执法）|
| 4 | 地名 / 机构名 / 行业术语 | 改革开放早期用"供销社"而非"便利店"、2000 年代初用"股份制"而非"有限合伙"|
| 5 | 物价 / 收入 / 生活水平 | 米价 / 工资 / 房价 / 通讯费符合年代锚点（可查历史新闻 / 统计公报）|
| 6 | 通讯 / 信息传递方式 | 1990s 用 BP 机 + 公用电话；2000s 前期用短信 + 固话；2010 前后用 QQ；手机普及曲线要对 |
| 7 | 交通 / 支付方式 | 绿皮车 / T 字头 / 高铁的覆盖年代；银行转账 / 支付宝 / 微信支付的普及时间 |

## 语言铁律

- 人物内心独白必须口语化 / 直觉化，禁止商业分析 / 博弈论术语渗入叙事
- ✗"他迅速分析了当前的债务状况" → ✓"他把那叠皱巴巴的白条翻了三遍"
- ✗"信息落差就在这儿" → ✓"他们不知道的，他知道"
- ✗"以这种人的性格，这时候不会撕破脸" → ✓直接写对方的行为反应
- ✗"他核心动机是拿回控股权" → ✓直接写他抢签名那一刻手在抖
- 法律 / 商业术语必须匹配设定年代的真实语感：2003 年民间借条不会写"逾期处置授权"，而是"到期没还，房子归乙方处理"
- 主角的判断通过行动和对话体现，不通过上帝视角的分析段落
- **禁止塞满术语**：投资 / 融资 / 股权 / 杠杆 / 对赌 / LP / GP 在单章累计不超过 3 次（读者不是 MBA）

## 叙事指导

以商战、社交博弈和信息差驱动剧情。
权力来自人脉、资本、信息和制度位置，不来自武力。
冲突解决靠谈判、交易、威慑、法律手段和利益交换。

钱权必须落地，通过物、势、地位变化和小人物反应兑现爽点。
人物关系网是核心资产，每次社交互动都应有利益计算。
主角需要保留非功能性时刻：思考、犹豫、社交润滑。
主角不是全知全能，必须在前 5 章内至少出现一次判断失误或信息偏差。

时代厚重感、人情债与制度摩擦是都市文的灵魂。
用场面、气味、动作、交易、压迫感切入，不要历史课件式开头。
嵌入 1-2 个时代锚点（物价、新闻事件、流行用语）增强年代沉浸感。
