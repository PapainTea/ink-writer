## 题材禁忌

- 无逻辑的巧合推进剧情（巧合最多用一次，且不能解决主要冲突）
- 配角降智配合主角（反派 / 队友 / 对手都必须按自己的利益与信息做判断）
- 无铺垫的高潮（爆点 / 转折必须有前 1-3 章的线索伏笔）
- 情节由作者意志驱动而非人物动机驱动（"主角必须 X 才能推进"是危险信号）
- 跳过合理反应（重大事件后角色应有情绪消化 / 后续决策，不能下一段就切日常）
- 核心冲突拖延解决（3 章未给反馈的冲突等于死冲突）
- 信息鸿沟无解（角色知道但读者不知道或反之，必须在 3-5 章内解释）

## 语言铁律

- **显示优先于陈述**：用动作 / 感官 / 细节落地情绪，不用抽象形容。✗"她很生气" → ✓"她把筷子拍在桌上，没再动那碗饭"
- **禁止作者说教**：不要在正文夹带"这告诉我们"/"由此可见" 类总结句，事件本身应该说话
- **禁止心理旁白灌水**：人物内心活动服务于决策或冲突，不作为凑字数的填充
- **同义渲染不超过两轮**：同一情绪 / 氛围 / 意象连续渲染最多 2 段，第 3 段必须切入新信息或新动作
- **段落节奏**：手机阅读友好，每段 3-5 行（80-150 字），长对话 / 动作段可更短
- **禁止清单式列举**：罗列物品 / 信息 / 动作时必须带入角色判断或取舍（"她数了数，三张纸币，两张皱了" 而非 "钱包里有三张纸币"）

## 检查点清单（写章时对照）

| # | 检查项 | 触发动作 |
|---|-------|---------|
| 1 | 本章是否推进了至少一项（信息 / 关系 / 处境 / 资源 / 决策）？| 若全无，Step 3 规划失败，回 Step 3 |
| 2 | 核心冲突是否在本章 / 最近 3 章内给过反馈？| 若未给，下一章必须兑现 |
| 3 | 是否有巧合推进关键情节？| 若有且超过 1 处，重写那一处改为因果 |
| 4 | 是否出现配角降智 / 反派排队送死？| 若有，重写反派视角的决策逻辑 |
| 5 | 情绪 / 氛围是否用抽象词直写？| 若有"愤怒""悲伤"类直写，改显示 |
| 6 | 心理旁白是否推进决策？| 若纯填充（超过 2 段且无决策 / 冲突），砍掉或改外化 |

## 叙事指导

- **根据作者意图调整叙事重心**：生活流偏情绪与关系，冒险流偏目标与障碍，悬疑流偏信息与推理
- **保持因果逻辑链完整**：A → B → C 必须可追溯，读者回翻能找到每步的前置
- **人物行为由动机驱动**：动机来源可以是欲望 / 恐惧 / 责任 / 信念 / 习惯，但必须在角色档案或前文中可查
- **每个场景至少推进一项**：信息 / 关系 / 处境 / 资源 / 决策 / 情绪状态变化，六选一
- **小冲突尽快兑现**：日常冲突（口角 / 小障碍 / 小决策）在当章或下一章给反馈；大冲突（跨卷主线）按卷纲推进
- **核心对手有脑子**：反派 / 竞争者必须有试探 / 误判 / 反扑，不能被单向输出
- **信息边界尊重**：角色只知道他知道的事，不得因作者视角泄露未知信息
