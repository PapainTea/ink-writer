## 题材禁忌

- 恐怖源头过早完全暴露（未知才恐怖；至少到第 1/3 篇幅才允许部分揭示）
- 主角无脑刚正面解决一切
- 用打脸 / 升级等爽文套路替代恐怖氛围
- 恐怖元素与日常场景割裂（好的恐怖来自日常的扭曲）
- 角色面对恐怖事件完全不害怕（过度镇定 = 反人类）
- 用大量血腥描写替代心理恐惧（血腥是副作用不是主武器）
- 跳级触发：从第一层直接跳到第三 / 四层（必须递进）
- 恐惧反应类型化（所有角色都"汗毛立起"），个体差异丢失

## 恐惧层级（感官指标清单 · 核心）

每一层级用**具体感官事实**而非情绪标签落地。Writer 在该层级写作时，至少命中该层级下 **2 个** 以上感官维度：

| 层级 | 状态 | 典型感官指标 | 触发下一层的标志 |
|------|------|------------|---------------|
| 1 不适感 | 微妙错位 / 违和 | 视觉（东西位置不对、光线色温异常）/ 听觉（背景声音缺失或多出一层）/ 嗅觉（熟悉空间多出一缕气味）| 角色开始主动核对 / 回头二看 |
| 2 不安 | 确认有异常但看不清全貌 | 触觉（汗、后颈起立、手指发凉）/ 平衡觉（站不稳、头重脚轻）/ 听觉（心跳盖过环境音）/ 呼吸（主动屏息 / 浅促）| 角色开始规避 / 改变动线 |
| 3 恐惧 | 威胁明确化，逃生本能启动 | 肌肉（腿软 / 指节发白）/ 视觉（视野窄化、周边模糊）/ 口腔（舌根发苦、口干到吞咽）/ 动作（失态）| 规则被打破 / 安全区失守 |
| 4 绝望 | 规则失效，安全彻底崩塌 | 思维断片（短句、主语丢失）/ 感官过载或钝化（听见自己骨骼响 / 或一片死寂）/ 生理失禁 / 语言原始化（只剩名词） | 高潮 / 喘息 / 新一轮循环 |

**跳级禁止**：1→3 或 2→4 之间必须有一章过渡；若情节需要急促，也要用 1-2 段做层级过渡。

## 满足类型触发条件（何时可以兑现爽点）

| 类型 | 触发前提 | 触发信号 |
|------|---------|---------|
| 真相揭示 | 至少 3 条线索已埋 + 其中至少 1 条被角色主动质疑过 | 角色拼合两条旧线索发现新结论 |
| 规则发现 | 角色已被规则惩罚 1 次 + 目睹他人被同规则惩罚 1 次 | 角色总结规则并主动验证 |
| 成功逃脱 | 至少经历过 1 次层级 3 以上的恐惧 + 付出代价（受伤 / 失物 / 同伴） | 抵达新安全区但立即产生新不适感（循环） |
| 反杀怪物 | 只在后半段允许 + 需要利用前文发现的规则，不许纯战力输出 | 规则验证成功即反杀成立 |
| 谜团解开 | 主要角色中至少 1 人付出了信息外的代价（身心创伤 / 关系破裂 / 信念崩塌）| — |
| 同伴获救 | 前文铺垫过该同伴的个人细节（书包 / 手机壳 / 口头禅 / 家庭片段）| 获救场景必须提及那个细节 |

## 语言铁律

- 恐怖用事实传达，不用情绪标签。✗"他感到一阵恐惧" → ✓"他后颈的汗毛一根根立起来"
- 禁止过度解释恐怖。异常现象只需呈现，不需叙述者出来总结"这一切都太不正常了"
- **克制叙事**：越恐怖越冷静。句子随恐惧升级而变短，但叙述者语气始终平稳。层级 3-4 平均句长应比层级 1-2 短 30% 以上
- 被淘汰 / 伤害的配角必须有至少一个暗示其个人故事的细节（书包里的补习班收据、手机壳上的贴纸、随身带的保温杯），让淘汰有重量
- **禁止感官套话**："汗毛立起""毛骨悚然""心跳加速"全章累计 ≤ 2 次（已列入 fatigueWords）
- **视角一致**：第一 / 第三人称固定后，感官描写必须来自该视角的身体（不要写"隔壁房间传来声音"除非角色能听见）

## 检查点清单（写章时对照）

| # | 检查项 | 触发动作 |
|---|-------|---------|
| 1 | 本章属于哪个恐惧层级？| 未确定 → 回 Step 3 补；章内最多跨 1 级 |
| 2 | 本章至少命中该层级 2 个感官维度？| 未命中 → 扩写补感官细节 |
| 3 | 有无"情绪标签直写"（恐惧 / 害怕 / 震惊 / 绝望）？| 有 → 改为感官事实 |
| 4 | 有无作者出来解释异常？| 有 → 删解释保留呈现 |
| 5 | 配角受害 / 淘汰时有无个人细节铺垫？| 无 → 前文补 1 处或本章补当下细节 |
| 6 | 安全区是否"到此为止"？| 若安全区持续超过 2 章未被打破 → 下一章必须破 |
| 7 | 是否连续 3 章重复同一恐怖模式？| 是 → 第 3 章必须打破模式（规则矛盾 / 可信来源说谎 / 安全区失效）|

## 叙事指导

氛围是第一生产力。用五感细节（声音、气味、温度、触感、光线）建立不安。
恐怖来自对未知的恐惧，信息揭示要克制。
"看不见的"永远比"看见的"更可怕。

角色的恐惧反应必须真实：颤抖、口干、思维混乱、判断力下降。
求生本能驱动行为，不是英雄主义。
每个安全区都是暂时的，喘息之后是更深的恐惧。

规则感：恐怖世界有自己的规则，发现规则是生存的关键。
信息管理：读者知道的和角色知道的之间的差距制造悬念。
日常的扭曲比凭空出现的怪物更恐怖。
每 3 章必须打破一次已建立的模式（规则矛盾、可信来源说谎、安全区失效），避免机械重复。
