## 题材禁忌

- 主角为推剧情突然仁慈、犯蠢
- 修为无铺垫跳跃式突破
- 法宝凭空出现解决危机
- 天道规则前后矛盾
- 用"大道无形""天道感应"跳过具体修炼过程
- 同质资源不写衰减默认全额结算
- 风格混入都市腔、游戏系统播报腔

## 修炼规则

- 境界突破必须有积累过程：悟道、丹药、战斗领悟、机缘
- 同质资源重复炼化必须写明衰减
- 法宝体系分品级，使用有代价（灵力、寿元、因果）
- 金手指/功法四维约束：
  - 能力上限：有明确的境界/品阶天花板
  - 附加代价：修炼/使用伴随代价（寿元、因果、心魔）
  - 触发条件：突破/觉醒需要特定条件（悟道、机缘、天劫）
  - 成长路径：功法随修为递进，不可跳阶获得
- 天道规则一旦设定不可违反，除非有明确的特殊机制
- 期初修为/资源从账本取，增量逐笔列出
- 跨大境界突破需要天劫或特殊条件

## 叙事指导

修炼与悟道是叙事核心，但必须融入剧情而非独立说教。
悟道场景用五感描写，不用抽象哲理灌输。
仙侠世界的规则感要强：因果、天劫、气运都是叙事工具。

人情债与道义约束是仙侠特有的驱动力。
门派政治、宗门博弈是重要的布局手段。
战斗以法术、法宝、阵法为核心，注重空间感和规模感。

## 因果 / 气运落地规则（可观察信号）

因果和气运是仙侠核心，但不能写成玄而又玄的抽象名词。每次因果 / 气运变动必须通过**至少一类可观察信号**落地，读者与其他角色能"看见"其变化：

- **物的反应**：随身法器 / 本命物品 / 功德簿 / 气运罗盘 / 因果秤 —— 光暗、重量、裂纹、鸣震、颜色偏移
- **境的反应**：风起、落雨、雾聚、花凋、鸟散、气压变化、灵气浓淡、结界涟漪
- **人的反应**：旁观修士脸色变化、同行者身上的业火 / 金光显隐、宿敌突然心悸、亲缘感应
- **宿命道具反应**：姻缘线、三生石、业镜、愿力牌、宿命牌局的具体落子变化
- **自身体感**：心口发热 / 发冷、识海清浊、经脉涩畅、发梢静电、寿元感知

## 功德 / 业力数值化轮廓

不要求给精确数字，但必须有**分级 + 典型触发**，避免 LLM 每章随意拉高拉低：

- **功德**：微（救小兽 / 解一疾）→ 小（解一村之厄）→ 中（除一方妖 / 传一脉法）→ 大（立宗门 / 平魔潮）→ 天（补天 / 续道）。功德上浮必须有具体受益者姓名或明确群体
- **业力**：污（欺弱）→ 浊（夺人机缘）→ 重（杀无辜）→ 深（灭门 / 断人道统）→ 煞（逆天绝地）。业力累积必须可追溯到具体事件
- 单章内功德 / 业力变动不得跨越两级，除非有渡劫 / 了愿 / 偿命等专属机制
- 气运盈亏与功德 / 业力**不等同**：气运可凭机缘 / 宝物 / 血脉获得，而功德 / 业力只能通过行为累积

## 因果 / 气运检查点

写完本章后自查：

- [ ] 本章涉及的因果 / 气运变化，是否至少有 1 类可观察信号落地？
- [ ] 功德 / 业力的增减，是否有具体事件、具体受益者 / 受害者？
- [ ] 是否用了"天道感应""大道无形""命数使然"这种跳过过程的抽象词而没有落地信号？
- [ ] 本章的因果是否在 3-5 章内有兑现铺垫（或已在旧账上清理）？
- [ ] 气运消涨是否与本章具体机缘 / 代价对得上（不是凭空波动）？
